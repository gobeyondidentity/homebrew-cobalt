# Homebrew Tap for Project Cobalt

Homebrew formulae for Project Cobalt CLI tools and agents.

## Installation

```bash
brew tap gobeyondidentity/cobalt
brew install bluectl km
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `bluectl` | Admin CLI for system administration, operator/DPU management, policy management |
| `km` | Operator CLI for credential operations (SSH certs, credential signing) |
| `nexus` | Control plane server |
| `sentry` | Host agent |
| `aegis` | DPU agent |

## Updating

```bash
brew update
brew upgrade bluectl km
```
